       *@return:
      **/
2. 变量
   2.1 变量名
       hello_world
   2.2 注释
       hello_world; /*  */

4. 条件语句

   }
##  学C++的垃圾话
1. 把C++当成一门新的语言学习(和C没啥关系!)。
2. 不要漏掉书中任何一个练习题–请全部做完并记录下解题思路。
3. 绝不要因为程序“很小”就不遵循某些你不熟练的规则。
4. 每学一个新知识都写一个demo。
5. 请不要做浮躁的人。
